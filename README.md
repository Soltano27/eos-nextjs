# EOS Foundation Website

Official website repository for the The Emmanuel Olatunde Sanya Foundation (EOS Foundation).

## About EOS

The Emmanuel Olatunde Sanya Foundation is a Nigerian nonprofit organization advancing youth brain health, neurological care access, digital resilience, health education, and community-driven innovation across Africa.

This website serves as the digital platform for EOS initiatives, programs, research communication, storytelling, partnerships, and public engagement.

## Tech Stack

- Next.js 15
- React
- TypeScript
- CSS Modules / Global CSS
- App Router Architecture

## Features

- Modular section-based architecture
- Responsive design
- Research and media pages
- Community health initiative pages
- Program showcase sections
- Optimized image assets
- Scalable nonprofit digital infrastructure

## Local Development

Clone the repository:

```bash
git clone https://github.com/Soltano27/eos-nextjs.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```
